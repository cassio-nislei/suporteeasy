✨ RECURSOS AUTOMÁTICOS:
═══════════════════════════════════════════════════════════════════════════

Durante a instalação, o sistema automaticamente:

✓ Cria tarefa agendada (Scheduled Task)
  → Executa no login de cada usuário
  → Privilégios elevados (Admin)
  → Auto-restart a cada 1 minuto se falhar

✓ Configura Firewall Windows
  → Regra inbound criada
  → Porta 37609 (bridge) configurada
  → Apenas tráfego local/necessário

✓ Adiciona Defender Exclusion
  → Pasta de instalação excluída
  → Sem varredura de AV
  → Melhor performance

✓ Cria arquivos de configuração
  → agent-config.json com credenciais
  → install-manifest.json com metadata
  → Diretórios de logs/temp criados
